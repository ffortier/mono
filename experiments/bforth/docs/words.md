# Words Reference

## Stack Operations
- `.s` — Print stack
- `.` — Pop and print
- `dup` — Duplicate top
- `depth` — Stack depth

## Arithmetic and Comparison
- `+` `-` `*` `/` — Basic arithmetic
- `0=` — Is zero
- `=` — Equal
- `>` `<` `>=` `<=` `<>` — Comparisons

## Memory/Variables
- `!` — Store
- `@` — Load
- `allot` — Allocate memory
- `here` — Memory pointer
- `constant` — Define constant
- `variable` — Define variable

## Output
- `."` — Print string

## Control Flow
- `begin` `until` — Loop
- `do` `loop` — Counting loop
- `if` `else` `then` — Conditional
- `while` `repeat` — Loop with condition
- `:` — Define word

## System
- `bye` — Exit
- `exit` — Return from word
